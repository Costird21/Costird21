### Daniel Radu

Building [TradeWolk](https://tradewolk.com) — an iOS trading-intelligence platform with read-only broker integrations, behavioural insights, and an AI trade journal.

#### Selected public work

- **[mt5-self-hosting-case-study](https://github.com/Costird21/mt5-self-hosting-case-study)** — write-up on running MetaTrader 5 terminals headless on a Windows VPS: silent broker-session disconnects, the multi-process bridge architecture, and the failure modes you only hit in production.
- **[mt5-watchdog](https://github.com/Costird21/mt5-watchdog)** — small Python library extracted from the case study. Detects the "process alive but MT5 disconnected" failure mode and reconnects with consecutive-failure tracking. MIT.
